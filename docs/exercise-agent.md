# Qadam Exercise Agent

## Задача

Агент не ждёт, пока каталог буквально закончится. Раз в сутки он проверяет запас активных и ещё не показанных упражнений отдельно для каждого направления и уровня. Если запас ниже порога, агент создаёт новую партию черновиков.

## Конвейер

1. Планировщик запускает инвентаризацию (`npm run exercise:inventory`).
2. Монитор считает запас по каждому сочетанию `category + level` и формирует очередь дефицитных сегментов.
3. Генератор получает профиль сегмента, список существующих упражнений и запрещённых повторов.
4. ИИ возвращает только структурированный JSON: цель, материалы, фразу родителя, три шага, критерий успеха, упрощение, усложнение, пользу и правило остановки на казахском и русском.
5. Валидатор проверяет обязательные поля, возраст 2–7 лет, длительность, уникальность, отсутствие медицинских обещаний, наказания, давления, опасных предметов и сенсорной перегрузки.
6. Семантический поиск отбрасывает близкие дубли.
7. Упражнение сохраняется со статусом `draft`. Ответственный специалист получает уведомление.
8. После одобрения статус меняется на `active`; только после этого упражнение участвует в персональных планах.

## Правило запуска

Рекомендуемый стартовый порог — минимум восемь активных вариантов на каждый уровень каждого направления. Дополнительно агент должен запускаться, если у конкретного ребёнка осталось меньше пяти ещё не показанных подходящих упражнений.

## Системная инструкция генератора

```text
Ты — редактор безопасных домашних развивающих упражнений Qadam для родителей детей 2–7 лет.
Создавай игровые действия длительностью 3–5 минут, которые можно выполнить дома без специального оборудования.
Не ставь диагноз, не обещай лечение или гарантированный результат. Не используй принуждение, удерживание ребёнка, наказание, пищу как обязательную награду, мелкие опасные предметы или сенсорную перегрузку.
Каждое упражнение должно отличаться от переданного каталога по цели, механике и фразе родителя.
Верни один JSON-объект по заданной схеме на естественном казахском и русском языках.
```

## Контроль публикации

Автоматически публиковать сгенерированный материал нельзя: продукт предназначен для детей, поэтому финальное решение остаётся за специалистом. Автоматическими остаются обнаружение дефицита, генерация, проверка формата, поиск дублей и постановка в очередь на ревью.

## Переменные

- `QADAM_MIN_READY_PER_LEVEL` — минимальный запас на сегмент, по умолчанию `8`.
- `QADAM_GENERATION_BATCH` — минимальный размер партии, по умолчанию `5`.
- Ключ модели хранится только в секретах серверной функции и никогда не попадает в браузер.
